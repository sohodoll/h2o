# H2O: a CRM

## Features

- Extendable table
- Basic Navigation
- Navbar
- Pagination
- Sort (by ID, count number)
- Filter by search prompt (for IDs)
- Deploy link: https://h2os.netlify.app/
- Development branch: https://github.com/sohodoll/h2o/tree/develop

## Tech Stack

TS, React, Vite, Tailwind, PostCSS
